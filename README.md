# Spam Mail Prediction using Machine Learning

## Overview

This project builds a machine learning model to classify emails as spam or ham (not spam). It is a binary classification problem that uses natural language processing techniques to analyze email content.

---

## Dataset Information

* Dataset: Spam Mail Dataset
* Features: Email text data
* Target:

  * 0 → Ham (Not Spam)
  * 1 → Spam

The dataset contains labeled email messages used to train and evaluate the model.
